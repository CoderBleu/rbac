# rbac

[Demo 演示](http://43.226.35.120:8089/)，maven打包项目，挂载服务器

## 技术栈
前端：Layui框架，Jquery，JS
后端：Springboot，Mybatis
数据库：Mysql 8.0，[数据库源代码](./rbac.sql)
